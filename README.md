# Exchango
Exchango is a currency exchange rate monitoring app designed to help you make the most of your currency exchanges. Set your desired threshold for the CNY to EUR exchange rate, and Exchango will notify you when it's the right time to exchange.

## Features
Real-time Exchange Rates: Get up-to-date CNY to EUR exchange rates from reliable sources.
Custom Threshold Alerts: Set a threshold rate, and receive notifications when the exchange rate falls below your set value.
User-Friendly Interface: Easy-to-use interface for setting up and managing your exchange rate alerts.
Push Notifications: Receive instant notifications on your mobile device when the exchange rate meets your criteria.
